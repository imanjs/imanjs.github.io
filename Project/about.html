Matchin' Markup



Matchin' Markup was created by Layke Jones as part of her Web Systems course at NCA&T.

Matchin' Markup is a tool to educate students on HTML and CSS fundamentals through engaging methods, such as a flashcard game.

Studying shouldn't be boring.
Matchin' Markup aims to support that.
